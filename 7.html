THE MANE ATTRACTION

The goal was to create a website on a subject that interested us, using Wordpress. I chose horse breeds and riding domains. Design was chosen on a Bootstrap, and I decided to continue with this black / white / turquoise theme. This project allowed me to progress a lot in PHP and Wordpress. The breed posts had to be linked to the most suitable riding domains and the disciplines should present the breeds that are best suited (related posts).

Type : School Project
Created : Second year of MMI
Duration : One months
Teamwork : No

Languages Used

HTML, CSS, JS, PHP
